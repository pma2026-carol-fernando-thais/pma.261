    Store Platform¶

    Plataforma de e-commerce baseada em microsserviços desenvolvida na disciplina PMA.261 — Plataformas, Microsserviços, DevOps e APIs.

    Stack¶

    Categoria Tecnologias
    Backend Java 25 (Spring Boot 4.x), Java 21 (Spring Boot 3.x), Python 3 (FastAPI)
    Banco de dados PostgreSQL + Flyway
    Containerização Docker + Docker Compose
    Orquestração Kubernetes — AWS EKS
    CI/CD Jenkins + OWASP Dependency Check
    Observabilidade Prometheus + Grafana
    Load Balancing Nginx + ALB (EKS) + HPA
