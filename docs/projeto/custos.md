# Cost Analysis

## Infraestrutura Utilizada

### AWS EKS

Cluster Kubernetes hospedado na AWS.

### Docker Hub

Armazenamento das imagens Docker.

### PostgreSQL

Banco de dados utilizado pelos serviços.

## Custos Considerados

- Nós do cluster Kubernetes
- Armazenamento
- Transferência de dados
- Imagens Docker

## Estratégias de Otimização

- Uso de containers leves
- Controle de réplicas
- Limites de CPU e memória
- Escalabilidade sob demanda
