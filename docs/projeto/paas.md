# PaaS

## Conceito

Platform as a Service (PaaS) fornece uma plataforma completa para desenvolvimento e execução de aplicações.

## Utilização no Projeto

Foram utilizados serviços gerenciados para facilitar o provisionamento da infraestrutura.

## Benefícios

- Menor esforço operacional
- Escalabilidade
- Alta disponibilidade
- Facilidade de manutenção

## Serviços Utilizados

- AWS
- Docker Hub
- GitHub
- Jenkins
