# Order API

## Objetivo

Desenvolver uma API responsável pelo gerenciamento de pedidos da plataforma.

## Tecnologias

- Java 21
- Spring Boot
- OpenFeign
- PostgreSQL

## Funcionalidades

- Criação de pedidos
- Consulta de pedidos
- Integração com Product Service
- Integração com Exchange Service

## Estrutura

- Controller
- Service
- Repository
- DTOs
