# Product API

## Objetivo

Desenvolver uma API responsável pelo gerenciamento de produtos da plataforma.

## Tecnologias

- Java 21
- Spring Boot
- Maven
- PostgreSQL

## Funcionalidades

- Cadastro de produtos
- Consulta de produtos
- Atualização de produtos
- Remoção de produtos

## Estrutura

A API foi desenvolvida utilizando arquitetura em camadas:

- Controller
- Service
- Repository
- Model
